DOM (Document Object Model o Modelos de Objeto de Documento)
El DOM nos permite que lenguajes como JS acceder, cambiar, añadir o eliminar contenido, estilos y estructura de un sitio web de forma dinámica tras su carga. 

Es una interfaz de programación que representa páginas HTML o XML como una estructura de árbol en memoria.
